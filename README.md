# Audio Transcriber Pipeline

Automated transcription of audio files using Playwright and Google AI Studio.

- [Technical Specification](docs/specification.md)

## Quick Start
1. Launch Chrome with debugging: `google-chrome --remote-debugging-port=9222`
2. Run transcription: `python transcribe.py your-audio.mp3`
